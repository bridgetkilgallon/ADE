Percent filter

A filter that appends a percent sign to a valid number.

  • Percent 34: {{ 34 | percent }}
  • Percent string '34%': {{ '34%' | percent }}
  • Invalid Percent "abc": {{ "abc" | percent }}

Percent editing directive

A directive that makes a percent editable (uses the same directive as money, decimal and integer).

The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
