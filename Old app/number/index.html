Number Filter

A filter that displays a number. This is a wrapper for Angular's built in filter so that we can display invalid inputs better.

  • Integer: {{ 1234 | flexnum}}
  • Decimal: {{ 1234.567 | flexnum}}
  • Integer string: {{ "1234" | flexnum}}
  • Invalid integer "integer": {{ "integer" | flexnum }}

Number editing directive

A directive that makes a number editable. It can be either an integer or decimal, it doesn't enforce any constraints. For display purposes, decimals will be rounded to 3 decimal places.

The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
